\section{MQAR Results}
\label{app:mqar}

Table~\ref{tab:mqar} presents the full Multi-Query Associative Recall
(MQAR) results.
Accuracy is 0\% across nearly all conditions, with only sporadic
non-zero values (10\% for Baseline and GRM at sequence length 128 with
8~key-value pairs).

As discussed in Section~\ref{sec:results:mqar}, this null result
reflects the base model's inability to parse the synthetic MQAR format,
not a failure of the quantization or memory-caching implementation.
The original Memory Caching paper~\cite{behrouz2026mc} trains models
from scratch on MQAR data; GPT-2 Medium was not trained on this task.
