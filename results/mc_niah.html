Needle-In-A-Haystack Memory Caching Evaluation

Comparing Standard FP16, Flat TurboQuant, Flat WHT, and our MC-TurboQuant Architecture

Original Contribution: MC-TurboQuant

We investigate a novel combination of two 2026 concepts: Memory Caching (arXiv:2602.24281) and TurboQuant (arXiv:2504.19874). While the Memory Caching paper proposes segmenting the sequence to bound memory and retrieve via Gated Residual Memory (GRM), it does not account for quantization of those cached checkpoints. MC-TurboQuant quantizes the completed segment memories using the WHT + asymmetric QJL quantizer, retaining the context of segments at ultra-low bit rates.
